# japan_stock_price

## Description

You can get japanes stock prices data on GUI

## Requirement

- Python 3.11.0

## Install

``` bash
pip install yfinance openpyxl pandas ndjson
```

## Usage

``` bash
git clone https://github.com/kk0128jp/japan_stock_price.git
$ cd japan_stock_price
$ python stock.py
```
